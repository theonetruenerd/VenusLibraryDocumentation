Step Codes
===================================

When a `.med` file is compiled into the corresponding `.hsl` file, Venus adds a
bunch of comments which are the metadata associated with things like what line the step
should be in the method.

One part of the metadata string is a UUID which corresponds to the step type in original Venus,
or sometimes just the library being used. Below is a (not-comprehensive) list of the UUIDs.
